# Data Science Toolbox
Project 1 for CS 1660, Fall 2020

This project contains various data science tools, each running as their own microservice through Docker.

## Progress
- [x] RStudio
- [x] Spyder
- [x] IBM SAS
- [x] Git
- [x] Jupyter Notebook
- [x] Orange
- [ ] Visual Studio Code
- [x] Apache Hadoop
- [x] Apache Spark
- [x] Tableau
- [x] SonarQube & SonarScanner
- [x] Tensorflow
- [x] Markdown
